Assistente de Voz Multi-Idiomas com Whisper e ChatGPT
Projeto desenvolvido no Google Colab.
Integra:
Gravação de áudio via navegador; Transcrição com Whisper; Geração de resposta com API da OpenAI; Síntese de voz com gTTS

Tecnologias Utilizadas:
Python 3; Google Colab ;JavaScript (MediaStream Recording API) ;Biblioteca whisper; SDK oficial da OpenAI; Biblioteca gTTS

Modelos Utilizados:
Whisper – Modelo de reconhecimento de fala da OpenAI; GPT-4o mini – Modelo de linguagem para geração de respostas

Funcionalidades:
Grava áudio diretamente do navegador; Transcreve automaticamente o áudio; Envia a transcrição para a API da OpenAI; Recebe resposta inteligente; Converte resposta em áudio novamente
